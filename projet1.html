Projet SMART CITY

Objectifs du projet

  • Créer un réseau privé d’objets connectés
  • Connecter différents objets connectés (OC)
  • Réaliser une communication entre les différents objets connectés
  • Stocker les données dans une base de données.
  • Mettre à disposition ces informations sur un site Web

Architecture Matérielle

Ma partie dans ce projet

Dans ce projet, ma partie était de créer le site web qui permet d’afficher des données récoltées par des objets connectés. Dans ce site web, se trouve une carte indiquant les coordonnées des objets connectés qui sont reliées à une base de données créée par un camarade qui s’occupe de cette partie. Le site a été créé en utilisant du HMTL/CSS et du JavaScript, j’ai pu utiliser notamment l’extension Vue pour mon site. J’ajoute à la suite le lien qui permet de consulter le site, depuis le site, mon code est accessible en inspectant la page pour plus d’information ou sur mon GitHub.
