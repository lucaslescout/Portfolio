Site WordPress

Lors de mon stage de BTS d’une durée de 7 semaines, j’ai pu travailler sur un site web en utilisant WordPress. J’ai donc utilisé l’affichage de leur site initial en y rajoutant principalement une visite guidée. Pour faire cette visite j’ai tout simplement utiliser des carrousels de photos disponible avec WordPress puis j’y ai ajouté, pour chaque zone du lycée, les photos correspondantes que je suis allé prendre avec mon smartphone. Sur le site se trouvent aussi d’autre fonctionnalités comme des raccourcis vers les sites qu’ils utilisent régulièrement et le menu du lycée est aussi consultable via mon site et il est mis à jour quand le personnel du lycée mette en ligne le PDF du menu. Ce site est consultable via le lien ci-dessous.

Sur le site, une fonctionnalité est aussi disponible, c’est le fait que le site soit une PWA (Progressive Web App) ce qui signifie que c’est une application web qui fonctionne comme un site web, mais qui peuvent apparaître à l'utilisateur de la même manière que les applications natives ou les applications mobiles. Le fonctionnement est expliqué via l’onglet application sur le site que j’ai aussi réalisé pour que cette fonctionnalité soit plus claire.
